## Installation


Install the dependencies and run the development server

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
## Tech Stack

**Client:**

## Run Locally

Clone the project

```bash
  git clone https://github.com/codestation21/ebuy.git
```

Go to the project directory

```bash
  cd my-project
```
Packages
```bash
npm i 
```
Run
```bash
npm run dev
```
